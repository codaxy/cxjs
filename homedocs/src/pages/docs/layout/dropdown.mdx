# Dropdown

The `Dropdown` widget is used for displaying content that appears below or above a trigger element.

**Import:** `import { Dropdown } from 'cx/widgets';`

## Basic Usage

A dropdown triggered by a button click.

<CodeExample
  code={`<Button onClick={(e, { store }) => {
  store.toggle("showDropdown");
}}>
  Toggle Dropdown
</Button>
<Dropdown
  visible={bind("showDropdown")}
  arrow
  offset={10}
>
  Dropdown content here.
</Dropdown>`}
>
  <DropdownBasic client:load />
</CodeExample>

## Search Suggestions

A dropdown that appears when a text field is focused.

<CodeExample
  code={`<TextField
  value={bind("query")}
  focused={bind("showSuggestions")}
  trackFocus
  placeholder="Search..."
/>
<Dropdown
  visible={bind("showSuggestions")}
  offset={1}
  matchWidth
>
  Suggestions appear here.
</Dropdown>`}
>
  <DropdownSearch client:load />
</CodeExample>

## Properties

| Property         | Type          | Default  | Description                            |
| ---------------- | ------------- | -------- | -------------------------------------- |
| `visible`        | `Sx<boolean>` | `false`  | Controls dropdown visibility           |
| `arrow`          | `boolean`     | `false`  | Shows an arrow pointing to the trigger |
| `offset`         | `number`      | `0`      | Distance from the trigger element      |
| `matchWidth`     | `boolean`     | `false`  | Match the width of the trigger         |
| `placementOrder` | `string`      | `"down"` | Preferred placement order              |
| `autoFocus`      | `boolean`     | `false`  | Focus first focusable element          |
