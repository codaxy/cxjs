# Range

The `Range` component highlights important zones on the chart.

**Import:** `import { Range } from 'cx/charts';`

## Example

<CodeExample
  code={`<Range x1={40} x2={80} colorIndex={10} name="Target Zone">
  <Text anchors="0 0.5 0 0.5" ta="middle">Target</Text>
</Range>
<Range y1={40} y2={70} colorIndex={4} name="Safe Range">
  <Text anchors="0.5 0 0.5 0">Safe</Text>
</Range>`}
>
  <RangesExample client:load />
</CodeExample>

## Properties

| Property     | Type         | Default     | Description         |
| ------------ | ------------ | ----------- | ------------------- |
| `x1`         | `Sx<number>` | `undefined` | Left bound          |
| `x2`         | `Sx<number>` | `undefined` | Right bound         |
| `y1`         | `Sx<number>` | `undefined` | Bottom bound        |
| `y2`         | `Sx<number>` | `undefined` | Top bound           |
| `colorIndex` | `number`     | `undefined` | Color palette index |
| `name`       | `string`     | `undefined` | Legend name         |
