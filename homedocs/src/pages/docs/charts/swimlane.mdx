# Swimlane

<ImportPath path="import { Swimlane } from 'cx/charts';" />

`Swimlane` creates individual swimlane bands where each one can have unique properties or content. Use with a `Repeater` to create swimlanes for each data record. Unlike `Swimlanes` (plural), this component gives you full control over each lane.

<CodeExample code={SwimlaneExampleCode}>
  <SwimlaneExample client:load />
</CodeExample>

## When to Use

- **Swimlane** (this) - When you need individual control over each lane (different colors, content inside lanes, custom styling per lane)
- **Swimlanes** - When you want automatic background bands for readability

## Configuration

| Property     | Type      | Default | Description                                      |
| ------------ | --------- | ------- | ------------------------------------------------ |
| `x`          | `string`  |         | Category value for vertical swimlane position.   |
| `y`          | `string`  |         | Category value for horizontal swimlane position. |
| `size`       | `number`  | `1`     | Swimlane width relative to category step (0-1).  |
| `vertical`   | `boolean` | `false` | Set to `true` for vertical swimlanes.            |
| `colorIndex` | `number`  |         | Background color from the palette.               |
| `style`      | `object`  |         | CSS style for the swimlane rectangle.            |
| `class`      | `string`  |         | CSS class for the swimlane element.              |
