# Legend

The `Legend` widget displays an index of chart elements.

**Import:** `import { Legend, LegendEntry } from 'cx/charts';`

## Example

<CodeExample
  code={`<Legend />
<Chart>
  <LineGraph name="Sales" data-bind="data" colorIndex={0} yField="sales" />
  <LineGraph name="Profit" data-bind="data" colorIndex={4} yField="profit" />
  <LineGraph name="Costs" data-bind="data" colorIndex={8} yField="costs" />
</Chart>`}
>
  <LegendExample client:load />
</CodeExample>

**Note:** Use `Legend.Scope` when multiple charts are on the same page.

## Properties

| Property   | Type      | Default  | Description     |
| ---------- | --------- | -------- | --------------- |
| `vertical` | `boolean` | `false`  | Vertical layout |
| `shape`    | `string`  | `"rect"` | Entry shape     |
