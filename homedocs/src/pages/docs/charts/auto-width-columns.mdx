# Auto Width Columns

<ImportPath path="import { Column } from 'cx/charts';" />

When displaying multiple column series side by side, `autoSize` automatically calculates column widths and positions. Column positions are determined based on previously seen data, so series maintain consistent placement across categories.

  <AutoWidthColumnsExample client:load />
</CodeExample>

## Configuration

| Property  | Type      | Description                                                                      |
| --------- | --------- | -------------------------------------------------------------------------------- |
| `autoSize`| `boolean` | Set to `true` to automatically calculate width and offset based on present data. |
| `uniform` | `boolean` | Set on `CategoryAxis` to ensure equal spacing regardless of data presence.       |
| `width`   | `number`  | Maximum width as a fraction of available space. Default is `0.5`.                |
