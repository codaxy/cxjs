# Bar

Individual bar element for custom bar charts with grouped or stacked data.

**Import:** `import { Bar } from 'cx/charts';`

## Example

<CodeExample
  code={`<Repeater records-bind="data" recordAlias="$point">
  <Bar colorIndex={0} name="2023" height={0.3} offset={-0.15} x-bind="$point.v1" y-bind="$point.category" />
  <Bar colorIndex={4} name="2024" height={0.3} offset={0.15} x-bind="$point.v2" y-bind="$point.category" />
</Repeater>`}
>
  <BarsExample client:load />
</CodeExample>

## Properties

| Property     | Type         | Default     | Description                  |
| ------------ | ------------ | ----------- | ---------------------------- |
| `x`          | `Sx<number>` | `undefined` | Bar length (x-axis value)    |
| `y`          | `Sx<string>` | `undefined` | Category position            |
| `height`     | `number`     | `0.5`       | Bar height (0-1)             |
| `offset`     | `number`     | `0`         | Vertical offset for grouping |
| `colorIndex` | `number`     | `undefined` | Color palette index          |
