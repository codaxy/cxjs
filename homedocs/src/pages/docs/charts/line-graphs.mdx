# LineGraph

Line charts are commonly used for data trends visualization.

**Import:** `import { LineGraph } from 'cx/charts';`

## Example

<CodeExample
  code={`<Chart axes={{ x: { type: NumericAxis }, y: { type: NumericAxis, vertical: true } }}>
  <Gridlines />
  <LineGraph name="Series 1" data-bind="points" colorIndex={0} area />
  <LineGraph name="Series 2" data-bind="points" colorIndex={4} yField="y2" />
</Chart>`}
>
  <LineGraphExample client:load />
</CodeExample>

## Properties

| Property     | Type        | Default     | Description              |
| ------------ | ----------- | ----------- | ------------------------ |
| `data`       | `Sx<Array>` | `undefined` | Array of data points     |
| `xField`     | `string`    | `"x"`       | Field for x-axis values  |
| `yField`     | `string`    | `"y"`       | Field for y-axis values  |
| `colorIndex` | `number`    | `undefined` | Color palette index      |
| `area`       | `boolean`   | `false`     | Fill area under the line |
| `line`       | `boolean`   | `true`      | Show the line            |
| `active`     | `boolean`   | `true`      | Enable interactivity     |
| `stacked`    | `boolean`   | `false`     | Stack with other series  |
