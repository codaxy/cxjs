# PieChart

Pie charts are commonly used to compare parts to the whole.

**Import:** `import { PieChart, PieSlice } from 'cx/charts';`

## Example

<CodeExample
  code={`<PieChart>
  <Repeater records-bind="slices">
    <PieSlice value-bind="$record.value" name-bind="$record.name" r={90} r0={40} colorMap="pie" />
  </Repeater>
</PieChart>`}
>
  <PieChartExample client:load />
</CodeExample>

## Properties

### PieChart

| Property     | Type     | Default | Description            |
| ------------ | -------- | ------- | ---------------------- |
| `angle`      | `number` | `360`   | Total angle of the pie |
| `startAngle` | `number` | `0`     | Starting angle         |

### PieSlice

| Property     | Type         | Default     | Description              |
| ------------ | ------------ | ----------- | ------------------------ |
| `value`      | `Sx<number>` | `undefined` | Slice value              |
| `r`          | `number`     | `undefined` | Outer radius             |
| `r0`         | `number`     | `0`         | Inner radius (for donut) |
| `colorIndex` | `number`     | `undefined` | Color palette index      |
| `colorMap`   | `string`     | `undefined` | Named color map          |
| `offset`     | `number`     | `0`         | Explode offset           |
