# MarkerLine

Marker lines highlight important values such as minimum or maximum.

**Import:** `import { MarkerLine } from 'cx/charts';`

## Example

<CodeExample
  code={`<MarkerLine y-bind="min" colorIndex={8}>
  <Text anchors="0 0 0 0" offset="5 0 0 5">Min</Text>
</MarkerLine>
<MarkerLine y-bind="max" colorIndex={4}>
  <Text anchors="0 0 0 0" offset="-5 0 0 5">Max</Text>
</MarkerLine>`}
>
  <MarkerLinesExample client:load />
</CodeExample>

## Properties

| Property      | Type         | Default     | Description                |
| ------------- | ------------ | ----------- | -------------------------- |
| `x`           | `Sx<number>` | `undefined` | Vertical line at x value   |
| `y`           | `Sx<number>` | `undefined` | Horizontal line at y value |
| `colorIndex`  | `number`     | `undefined` | Color palette index        |
| `legend`      | `string`     | `undefined` | Legend name                |
| `affectsAxes` | `boolean`    | `true`      | Affects axis range         |
