**Import:** `import { FieldGroup } from 'cx/widgets';`

The `FieldGroup` widget is a pure container which allows disabling of multiple form elements using a single property. `ValidationGroup` widget may be used for this purpose too, under the hood, `FieldGroup` is just an alias for it.
