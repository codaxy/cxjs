# insertElement

<ImportPath path="import { insertElement } from 'cx/data';" />

`insertElement` inserts one or more elements at a specific position in an array.

<CodeExample code={InsertElementExampleCode}>
  <InsertElementExample client:load />
</CodeExample>

## Signature

```ts
function insertElement<T>(array: T[], index: number, ...elements: T[]): T[]
```

## Parameters

| Parameter  | Type     | Description                               |
| ---------- | -------- | ----------------------------------------- |
| `array`    | `T[]`    | The array to insert into.                 |
| `index`    | `number` | Position to insert at.                    |
| `elements` | `T[]`    | Elements to insert.                       |

## Return Value

Returns a new array with elements inserted at the specified position.

## Examples

### Insert single element

```ts
const items = ["a", "b", "c"];
const result = insertElement(items, 1, "x");
// ["a", "x", "b", "c"]
```

### Insert multiple elements

```ts
const items = ["a", "b", "c"];
const result = insertElement(items, 1, "x", "y");
// ["a", "x", "y", "b", "c"]
```

### Insert at beginning

```ts
const items = ["a", "b", "c"];
const result = insertElement(items, 0, "x");
// ["x", "a", "b", "c"]
```

### Insert at end

```ts
const items = ["a", "b", "c"];
const result = insertElement(items, 3, "x");
// ["a", "b", "c", "x"]
```
